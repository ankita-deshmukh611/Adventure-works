# Adventure-works sales Analytics project
Developed an interactive Sales Analytics Dashboard using Power BI to analyze business performance and generate actionable insights from the Adventure Works dataset. The project involved data cleaning, data transformation, data modeling, and DAX calculations to track key KPIs including Sales, Revenue, Profit, Orders, and Return Rate. Built interactive visualizations with filters and slicers to explore trends across products, customers, and regions and support data-driven decision making.

Tools Used:

Power BI
Power Query
DAX
Data Modeling
Data Visualization
Excel

Key Skills:
Data Cleaning | KPI Analysis | Dashboard Development | Business Intelligence | Data Visualization | DAX | Reporting
